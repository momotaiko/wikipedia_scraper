프로그램 구상

위키피디아의 페이지 안에 있는 모든 단어를 외우기 편한 형태로 만들어 저장해두고 계속 볼 수 있는 목적의 앱 개발

핸드폰 앱으로 만들 예정(프론트엔드 flutter, 백엔드 python)

핸드폰에 위키피디아 페이지 링크를 넣으면 자동으로 영단어와 한국어 뜻을 대응시킨 텍스트 파일 생성
그 텍스트 파일을 앱에서 바로 열람할 수 있게 하기. (망각곡선, 단어 분류 등의 사용자 친화적 기능들은 일단 후순위로 두기)

- 구글 클라우드로 호스팅 준비(완료)
- 파이썬으로 백엔드 로직 구성하기
- flutter로 어플 프론트엔드 구성하기
